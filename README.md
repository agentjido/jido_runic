Runic workflow integration for the Jido ecosystem.

## Overview

JidoRunic bridges [Runic](https://github.com/zblanco/runic) DAG workflow composition with [Jido](https://github.com/agentjido/jido) agents, enabling powerful dataflow-driven agent workflows.

## Installation

Add `jido_runic` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:jido_runic, github: "agentjido/jido_runic"}
  ]
end
```
